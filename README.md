# 📈 Profit Analysis using SQL, Orange & Power BI

## 📌 Overview
Analyzed **50 startup records** to identify how **R&D, Marketing, Administration spending, and State** influence startup profitability. Developed a Multiple Linear Regression model to predict profit and built an interactive Power BI dashboard to support data-driven investment and business decisions.

## 🛠️ Tools Used
- MySQL
- Orange Data Mining
- Power BI
- Microsoft Excel
- Git & GitHub

## 📂 Project Pipeline
Startup Dataset (MySQL)
→ Data Validation & Quality Checks
→ Data Cleaning & Preprocessing
→ Multiple Linear Regression (Orange)
→ Model Evaluation & Profit Prediction
→ Power BI Dashboard
→ Business Insights & Recommendations

## 📊 Key Highlights
- Analyzed **50 startup records** across **3 states** to evaluate factors affecting business profitability.
- Built a **Multiple Linear Regression model** with an **R² score of approximately 0.944** for profit prediction.
- Generated profit forecasts for new startup investment scenarios using the trained regression model.
- Developed an **interactive Power BI dashboard** featuring KPIs, state-wise profit analysis, investment trends, and spending vs. profit relationships.
- Delivered actionable business recommendations to optimize **R&D, Marketing, and Administration** spending for improved profitability.
